Install locally

  • In your computer terminal

    git clone https://github.com/dahfazz/pugrace.git

  • Open and edit conf.js file

    port: '8880',       // The port you want to launch the game on
ip:   'localhost'   // The IP address you want to launch the game on

  • Install every project dependencies

    $ npm install
$ bower install

  • And finally, back in your terminal

    $ node app.js

> here we go on localhost:8880
